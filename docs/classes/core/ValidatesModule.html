    Class Method Details

    + (void) validates(attribute_name, opts)

    Specifies validation on a named attribute.

    Examples:

    requires a user_name: 

    class User extends core.model
  # ...
  @validates "user_name", presence: true, length: { min: 5 }

    Parameters:

    • attribute_name (String) — the name of the attribute to validate.
    • opts (Object) — a hash of the validators to invoke and options to pass to each.
